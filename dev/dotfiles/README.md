# Dotfiles
These are the dotfiles for my minimal Void Linux installation run on my main machine

| Application Type     | Name            |
|----------------------|-----------------|
| Distro               | Void            |
| Window Manager       | DWM             |
| Terminal Emulator    | Alacritty       |
| Application Launcher | dmenu           |
| Statusbar            | slstatus        |
| Editor               | Neovim          |
| File Manager         | Ranger          |
| Browser              | Qutebrowser     |
| Color Scheme         | [ColexDev](https://github.com/ColexDev/dotfiles/blob/main/.config/nvim/colexdev.vim)|
| Font                 | Source Code Pro |
| Wallpaper            | [Light Bulb](https://github.com/ColexDev/dotfiles/blob/main/misc/wallpaper.png)|
